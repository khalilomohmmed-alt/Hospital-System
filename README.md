# مستشفى الخليل - Hospital Dashboard

مرحباً! هذا مشروع ويب لمستشفى الخليل يتيح للمستخدمين حجز المواعيد ومتابعتها، بالإضافة إلى لوحة تحكم لعرض إحصائيات المواعيد لكل قسم.

## مميزات المشروع

- حجز المواعيد بسهولة مع إدخال الاسم، العمر، والقسم.
- عرض كل المواعيد المحجوزة مباشرة تحت الفورم.
- لوحة تحكم (Dashboard) تعرض:
  - عدد المواعيد الكلي.
  - عدد المواعيد لكل قسم (الأسنان، العيون، الباطنة، الأطفال).
  - آخر 5 مواعيد محجوزة.
- صفحة "عن الموقع" لشرح الهدف وطريقة الاستخدام.
- إمكانية ربط المشروع مع **قاعدة بيانات** مثل MySQL لتخزين المواعيد بشكل دائم ومشاركته بين عدة مستخدمين.

## التقنيات المستخدمة

- HTML
- CSS
- JavaScript
- (اختياري) PHP وMySQL لإضافة قاعدة بيانات

## كيفية الاستخدام

1. افتح المشروع على متصفحك من خلال فتح ملف `index.html`.
2. حجز المواعيد يتم مباشرة عن طريق الفورم.
3. للوصول إلى لوحة التحكم، افتح ملف `dashboard.html`.
4. لعرض نبذة عن الموقع، افتح ملف `about.html`.
5. لتفعيل قاعدة البيانات، يمكن تعديل ملفات PHP وربطها بقاعدة MySQL.
